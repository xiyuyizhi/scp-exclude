### installation

```
npm install scp-exclude -g
```

### Usage

similar with scp , with the capacity of exclude some local file or dir by option `-I or --ignore`

```
scp-exclude  localDir  root@host:remotePath  -I  node_modules,.vscode  -1  -P 8899
```
